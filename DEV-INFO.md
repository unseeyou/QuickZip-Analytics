# DEV INFO
## Setup Instructions
1. Download python from python.org
2. run `pip install uv` in the terminal
3. in the project folder once you have `git cloned` the files, run `uv sync`

pls send a message in the insta group if you need further help :)

## Contributing
1. Before making any changes please make a new branch
2. Implement your changes onto this branch
3. Once the changes are made, create a pull request into `main` from your branch
4. After someone reviews the changes, if they approve it then the changes will be reflected on the main branch. At this stage the branch in which you were adding the new change is now safe to be deleted or you can re-use it for another feature as long as it is still up to date.
